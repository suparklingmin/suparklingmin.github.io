*Introduction to Linguistic Statistics* · 한국언어학회 **2025 LSK-시몬느 언어학 학교** (2025년 8월 11–14일 · 실시간 ZOOM 웨비나)

통계학과 프로그래밍에 익숙하지 않은 언어학 연구자를 위한 입문 강의입니다. 통계 모델링과 통계적 추론의 이론적 기초를 다루고, Python 프로그래밍 실습을 병행합니다. (1시간 × 8회 · 총 8시간)

1. **Python 시작하기** — Google Colaboratory 소개, 수치 연산(NumPy)
2. **데이터 분석 준비** — tidy data vs. messy data, 변수의 유형
3. **데이터 탐색** — 기술통계량(pandas), 시각화(seaborn)
4. **통계 모델링 (1)** — 이론적 배경, 선형 회귀분석(statsmodels)
5. **통계 모델링 (2)** — 로지스틱 회귀분석, 푸아송 회귀분석
6. **통계 모델링 (3)** — 혼합 효과 모형, 모델 선택
7. **통계적 추론 (1)** — 가설 검정 이론, t-검정
8. **통계적 추론 (2)** — 카이제곱 검정, 분산분석
